#  THREADFLOW SHOWCASE: MITRE ATT&CK® Visualizer

*O **ThreadFlow Showcase** é uma plataforma interativa de visualização técnica baseada no framework **MITRE ATT&CK®**. O objetivo deste projeto é fornecer uma interface imersiva para a exploração das **14 táticas** utilizadas por adversários em campanhas de ataques cibernéticos. A aplicação detalha metodologias de **exploração de baixo nível**, **manipulação de memória**, **persistência** em sistemas operacionais e protocolos de **comando e controle (C2)**.*

---

##  Sobre o Projeto / About the Project

### 🇧🇷 Português
*Este projeto nasceu da necessidade de transformar dados brutos de cibersegurança em uma experiência visual compreensível. O framework MITRE ATT&CK® é a base mundial para entender como hackers operam, e o **ThreadFlow** atua como uma ponte entre a teoria complexa e a visualização prática.*

*   **O que é o MITRE ATT&CK?** 
    > *Imagine um manual que cataloga todos os métodos que um ladrão pode usar para invadir uma casa: desde pular o muro, desativar o alarme, até abrir o cofre. O MITRE faz exatamente isso, mas para o mundo digital, listando cada passo de um ataque hacker.*

### 🇺🇸 English
*This project was born from the need to transform raw cybersecurity data into a comprehensible visual experience. The MITRE ATT&CK® framework is the global foundation for understanding how hackers operate, and **ThreadFlow** acts as a bridge between complex theory and practical visualization.*

*   **What is MITRE ATT&CK?**
    > *Imagine a manual that catalogs every method a burglar could use to break into a house: from climbing the fence and disabling the alarm to cracking the safe. MITRE does exactly that, but for the digital world, listing every step of a cyberattack.*

##  Objetivos Acadêmicos / Academic Objectives

### 🇧🇷 Português
*   **Mapeamento de Ameaças:** *Demonstrar o fluxo lógico de um **ataque progressivo** (Kill Chain) em ambientes corporativos, permitindo visualizar como uma invasão evolui da busca por falhas até o roubo de dados.*
*   **Educação em Cibersegurança:** *Fornecer descrições técnicas detalhadas sobre **vulnerabilidades** e **vetores de ataque**, servindo como material de estudo para analistas de segurança.*
*   **Engenharia de Interface:** *Aplicar conceitos de **design de interface de terminal (TUI)** utilizando tecnologias web modernas para criar uma estética "hacker" funcional e contextualizada.*

### 🇺🇸 English
*   **Threat Mapping:** *Demonstrate the logical flow of a **progressive attack** (Kill Chain) in corporate environments, allowing users to visualize how a breach evolves from reconnaissance to data exfiltration.*
*   **Cybersecurity Education:** *Provide detailed technical descriptions regarding **vulnerabilities** and **attack vectors**, serving as study material for security analysts.*
*   **Interface Engineering:** *Apply **Terminal User Interface (TUI)** design concepts using modern web technologies to create a functional and contextualized "hacker" aesthetic.*

---

##  Tecnologias Utilizadas / Technologies Employed

*   **HTML5:** *Estruturação **semântica** de alta performance para garantir acessibilidade e SEO.*
*   **CSS3:** *Uso avançado de **Flexbox/Grid**, animações de **glitch**, efeitos de **scanlines** (linhas de monitor antigo) e total **responsividade** para dispositivos móveis.*
*   **JavaScript (ES6+):** *Lógica central para **manipulação de DOM**, efeito de digitação (**Typewriter**) para simular terminais reais e carregamento dinâmico de dados.*

##  Funcionalidades Técnicas / Technical Features

### 🇧🇷 Português
1.  **Navegação Granular:** *Acesso completo às **14 fases** do framework através de um menu dinâmico lateral.*
2.  **Cartões de Detalhamento:** *Cada tática possui **4 sub-categorias** que explicam desde a parte técnica (código) até o impacto no sistema.*
3.  **Simulação de Terminal:** *Interface que replica o visual de centros de operações de segurança (**SOC**), utilizando fontes mono-espaçadas e alto contraste.*
4.  **Mitigação Integrada:** *Além de mostrar o ataque, o projeto sugere como **defender** o sistema contra aquela tática específica.*

### 🇺🇸 English
1.  **Granular Navigation:** *Full access to the **14 phases** of the framework through a dynamic side menu.*
2.  **Detail Cards:** *Each tactic features **4 sub-categories** explaining everything from technical code to system impact.*
3.  **Terminal Simulation:** *Interface that replicates the look of Security Operations Centers (**SOC**), using monospaced fonts and high contrast.*
4.  **Integrated Mitigation:** *Beyond showing the attack, the project suggests how to **defend** the system against that specific tactic.*
